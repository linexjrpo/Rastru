Conectar ao Google Sheets
Banco de dados permanente para suas campanhas
Como configurar em 3 passos
1
Crie uma planilha no Google Drive → abra o Apps Script (Extensões → Apps Script)
2
Cole o código do Apps Script → Deploy → New deployment → Web App → Execute as: Me → Access: Anyone
3
Cole a URL do Web App abaixo e clique em Salvar
info A URL fica salva no localStorage do navegador. Cada membro configura a sua própria URL apontando para a mesma planilha.

Conectar ao Notion
Banco de dados permanente para suas campanhas
Como configurar em 3 passos
1
Acesse notion.so/my-integrations → crie integração "Rastru" → copie o Internal Integration Token
2
Crie um Database no Notion com as propriedades listadas abaixo. Clique em "..." → "Connect to" → selecione "Rastru". Copie o Database ID da URL
3
Cole token e ID abaixo → clique Salvar e Conectar
info Token salvo apenas no localStorage do navegador. Para equipe: cada membro configura seu token com acesso ao mesmo database.
